# Формулы и переменные

Автоматизация в Anima позволяет связывать виджеты друг с другом. Например, вы можете сделать так, чтобы бонус к атаке автоматически пересчитывался при изменении характеристики Силы.

## Как работают переменные
Переменная — это ссылка на значение конкретного поля виджета. Чтобы использовать значение в формуле, его нужно заключить в квадратные скобки: `[имя_переменной]`.

### Типы переменных
1.  **Собственные значения**. Вы можете обращаться к полям того же самого виджета, используя префикс `local-` и техническое имя поля. Например, `[local-value]` или `[local-current]`.
2.  **Связанные виджеты**. Чтобы получить данные из другого виджета, необходимо привязать его к текущему с помощью меню Связи.
2.  **Алиасы**. По умолчанию привязанные виджеты дают обращаться к переменным по их техническому id, например `modifier-663988958975`. Но вы можете задать для переменной алиас в настройках Данных, назвав переменную более осмысленно, например `str`.

## Примеры использования

### Автоматический расчёт модификатора
Если у вас есть счётчик, и для его поля «Текущее значение» задан алиас `str`, вы можете создать другой виджет, который будет вычислять модификатор по формуле:
`floor(([str] - 10) / 2)`

### Кнопка броска с бонусом
Для кнопки атаки можно задать формулу:
`1d20 + [str] + [proficiency_bonus]`
Здесь `str` и `proficiency_bonus` — это алиасы соответствующих виджетов на листе.

## Список доступных функций
В формулах можно использовать следующие математические функции:
*   `floor(x)` — округление вниз до ближайшего целого.
*   `ceil(x)` — округление вверх до ближайшего целого.
*   `round(x)` — округление к ближайшему целому.
*   `abs(x)` — модуль числа (абсолютное значение).
*   `min(x, y, ...)` — выбор минимального значения из перечисленных.
*   `max(x, y, ...)` — выбор максимального значения из перечисленных.
*   `sqrt(x)` — квадратный корень.

Полный список: [Нотация формул](/docs/notation.md).

## Полезные советы
*   **Регистр важен**. Убедитесь, что имя переменной в формуле точно совпадает с алиасом поля.
*   **Проверка ошибок**. Если в формуле допущена ошибка или переменная не найдена, виджет может отобразить значение `~error~` в меню Связи. Проверьте правильность написания алиасов и наличие скобок.
