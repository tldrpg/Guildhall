# Каталог виджетов

Виджеты — это строительные блоки вашего шаблона. Каждый виджет предназначен для определённого типа данных или действий.

## Основные виджеты

### Модификатор (Mod)
Специализированный виджет для числовых бонусов и штрафов, который позволяет бросать кости по клику.
*   **Применение**. Модификаторы характеристик (Сила, Ловкость), бонусы мастерства.
*   **Особенности**. Умеет автоматически добавлять знак `+` перед положительными числами (настраивается в Стилях).

### Текстовая строка (Input) и Текстовый блок (Textarea)
Используются для ввода текстовой информации.
*   **Текстовая строка**. Короткие строки (имя, класс). Текст уходит за край, если не помещается.
*   **Текстовый блок**. Длинные тексты (предыстория, способности, заметки). Текст переносится на новую строку, если не помещается.

### Чекбокс (Checkbox)
Простой переключатель состояния («включено» или «выключено»).
*   **Применение**. Владение навыком, наличие предмета.

### Счётчик — поле (Counter)
Предназначен для числовых значений, которые часто меняются.
*   **Применение**. Очки здоровья (HP), опыт, текущий запас золота.
*   **Особенности**. Может отображать текущее и максимальное значение через разделитель (например, `10/25`).

### Счётчик - чекбоксы (CounterDots)
Отображает значение в виде чекбоксов.
*   **Применение**. Ячейки заклинаний, уровни усталости, пункты удачи.
*   **Особенности**. Наглядное отображение ограниченного ресурса.
