# Начало работы

В этом разделе описаны первые шаги для начала использования Anima.

## Авторизация
Для работы с системой необходимо авторизоваться. [Anima](https://anima.longstoryshort.app/) и [Long Story Short](https://longstoryshort.app/) использют единую систему учётных записей. Вы можете авторизоваться, используя один из перечисленных методов: аккаунт Google, аккаунт Яндекс.

## Создание первого персонажа
Чтобы создать персонажа (лист персонажа), выполните следующие действия:

1.  **Выбор шаблона**. На главной странице перейдите во вкладку «Шаблоны». Здесь представлены шаблоны для различных игровых систем, как ваши, так и шаблоны сообщества.
2.  **Создание листа**. Нажмите «Превью» у выбранного шаблона и нажмите «Создать персонажа» в правом верхнем углу. Система создаст лист для вашего персонажа, связанный с выбранным шаблоном (дальнейшие изменения шаблона могут затем быть применены к листу персонажа).
3.  **Заполнение данных**. После создания вы попадете в интерфейс листа, где сможете начать заполнять характеристики и описание своего героя.
