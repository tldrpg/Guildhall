# Броски костей

## Как делать броски

### 1. Виджеты на листе
В большинстве шаблонов уже настроены кнопки для бросков (например, атака оружием или проверка характеристики). Просто нажмите на соответствующую кнопку или значение, чтобы выполнить бросок.

### 2. Дайсница
Дайсница позволяет совершать произвольные броски. Она находится в правом нижнем углу экрана.
*   Вы можете выбрать нужную кость (d4, d6, d8, d10, d12, d20, d100) кликом по иконке.
*   Вы можете ввести произвольную формулу в поле ввода.

## Лог бросков
Все результаты бросков отображаются в логе. В нём вы можете увидеть:
*   Итоговый результат.
*   Детальную расшифровку броска (какие значения выпали на каждой кости).

## Нотация бросков
Дайсница поддерживает сложный синтаксис бросков, вот некоторые примеры:

### Базовые броски
*   `2d6` — два 6-гранных кости.
*   `1d20 + 5` — бросок d20 с модификатором +5.
*   `(2d6 + 2) * 2` — результат выражения в скобках удваивается.
*   `1d20 + 5 (Атака)` - лейбл для броска.
*   `round(1d12 / 2)` — округление к ближайшему целому.
*   `4d6kh3` (оставить 3 лучших), `4d6dl1` (сбросить 1 худший).
*   `1d20cs=20cf=1` — критический успех на 20, критический провал на 1.

Полное описание возможностей смотрите в [Нотации формул](/docs/notation.md).
