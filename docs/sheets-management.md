# Работа с листами

Лист персонажа — это экземпляр шаблона, содержащий данные конкретного героя. Структура и расположение элементов (виджетов) в листе фиксированы и определяются шаблоном, на основе которого он был создан.

## Интерфейс листа
При открытии листа персонажа вам доступны следующие элементы управления, располагающиеся в шапке:

*   **Заголовок**. Здесь можно изменить имя персонажа (нажмите на текущее имя, чтобы активировать поле ввода).
*   **Оглавление**. По центру шапки приведён интерактивный список всех страниц, если их больше одной.
*   **Печать**. Иконка принтера позволяет подготовить лист к печати или сохранить PDF.
*   **Настройки**. Внутри находится включение редактирования виджетов, необходимое для изменения некоторых данных листа.
*   **Шаблон**. Информация о шаблоне, от которого был создан лист, а также применение обновлений, внесённых в шаблон.
*   **Индикация сохранения**. Иконка облака показывает статус синхронизации данных с сервером.

## Ввод данных
Вы можете взаимодействовать с виджетами на листе:
*   Заполнять текстовые поля и области.
*   Изменять значения счётчиков.
*   Отмечать чекбоксы и ячейки.
*   Нажимать на кнопки бросков кубиков.

Если виджет не поддерживает прямое изменение данных (к таким, например, относится виджет Модификатор), для изменения его данных необходимо включить режим редактирования через кнопку настроек в шапке.

Все изменения данных сохраняются автоматически. Для этого не требуется нажимать отдельную кнопку «Сохранить».

## Управление списком листов
На главной странице в разделе «Листы» отображаются все ваши персонажи. Каждая карточка листа позволяет:
*   **Перейти к листу**. Нажмите на название или карточку персонажа.
*   **Клонировать**. Создать точный дубликат персонажа со всеми заполненными данными.
*   **Удалить**. Безвозвратно удалить лист персонажа.
