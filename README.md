# Guildhall. Документация Project Paper

Guildhall — это раздел документации для пользователей Project Paper — конструктора листов персонажей для настольных ролевых игр.

## Основные разделы

1.  **[Начало работы](/docs/getting-started.md)** — авторизация и создание первого персонажа.
2.  **[Работа с листами](/docs/sheets-management.md)** — управление персонажами и режимы работы.
3.  **[Создание шаблонов](/docs/creating-templates.md)** — проектирование собственных листов персонажей и публикация в сообщество.
4.  **[Каталог виджетов](/docs/widgets.md)** — описание доступных компонентов шаблона.
5.  **[Формулы и переменные](/docs/formulas-and-variables.md)** — связывание виджетов и автоматизация вычислений.
6.  **[Броски костей](/docs/rolling-dice.md)** — система бросков, лог результатов.
7.  **[Нотация формул](/docs/notation.md)** — синтаксис бросков костей и математических выражений.
